Pothole Detection System

An end-to-end intelligent road monitoring system that automatically detects potholes using computer vision and artificial intelligence.

The system includes:

📱 Flutter mobile application
🌐 Flutter web application
🐍 Flask AI inference server
🐘 PHP REST API
🤖 YOLO-based pothole detection model
🗺️ Interactive map for pothole visualization
📊 Real-time statistics and analytics dashboard

Features:

Detect potholes from images
Detect potholes from recorded videos
AI-powered object detection using YOLO
Display detected potholes on an interactive map
Dashboard with road damage statistics
Store detection results in a database
Mobile and web interfaces
REST API communication between services
Model training pipeline for custom datasets

Tech Stack:

Flutter Mobile
Flutter Web
Python
Flask
PHP
YOLO
MySQL
